    # maxGraph

    # maxGraph是什么？

    • 官方网站:https://maxgraph.github.io/maxGraph/
      • maxGraph 是一个 TypeScript 库，可以显示矢量图并允许与矢量图交互。从高层次上讲，它提供：Node 和 Edge；
    • maxGraph其实是mxGraph的进阶版？我是这么理解的；目前mxGraph停止维护；可以直接看maxGraph.
      • draw.io的就是用这个绘制的

    # maxGraph我实现了什么？

    • 我使用它绘制了类似定制版绘图软件demo
      • 功能：
        • 1.可拖拽形状，图片，连接线形成拓扑图；
        • 2.设定可拖拽移动区域
        • 3.Graph可zoomIn/zoomOut；
        • 4.绘图可修改名称并不能重复；
        • 5.形状指定端口，不能重叠；
        • 6.实现坐标轴，原点在左下角
        • 导入文件初始化有默认灰色连接线，当形成新的拓扑连接（中间有n个连接点）灰色连接线消失；

      Shape.prototype.getPorts = function () {
    const numPorts = 8; // 8个端口，可以根据需要调整
    const ports = {};
  
    for (let i = 0; i < numPorts; i++) {
      const angle = (360 / numPorts) * i;
      const radians = angle * (Math.PI / 180);
      const x = Math.cos(radians) * 0.5 + 0.5; // 0.5是圆心x坐标，这里假设圆半径为0.5
      const y = Math.sin(radians) * 0.5 + 0.5; // 0.5是圆心y坐标，这里假设圆半径为0.5
  
      ports[`port${i}`] = {
        x: x,
        y: y,
        perimeter: true,
        constraint: `${angle} degrees`
      };
    }
  
    return ports;
  };

    • 因为需求需要将原点变更到左下角，以及整体放大会超出所需容器，所以使用api：

    这样鼠标放大缩小改变scale，通过计算将图像平移回原处

    let currentScale = 1;
function addMouseWheelZoom(graph: Graph, container: HTMLDivElement) {
  let scaleFactor = 1.1;
  let currentScale = 1;

  container.addEventListener('wheel', (evt: WheelEvent) => {
    evt.preventDefault();
    console.log(evt,'evt')
    const rect = container.getBoundingClientRect();
    const left = rect.left;
    const bottom = rect.bottom;
    // console.log(bottom,'bottom')
    const prevScale = currentScale;

    if (evt.deltaY > 0) {
      if(currentScale > 1){
         currentScale /= scaleFactor;
        }else{
          return
        }
    } else {
      // Zoom in
      currentScale *= scaleFactor;
    }
    console.log(prevScale,currentScale)
    // Calculate new translate values to keep the left bottom corner fixed
    const newTranslateX = (left / prevScale - left / currentScale);
    const newTranslateY = (bottom / prevScale - bottom / currentScale);
    // console.log( graph.view.translate.y,newTranslateY, graph.view.translate.y - newTranslateY,'newY')
    graph.view.scaleAndTranslate(
      currentScale,
      0,
      evt.deltaY < 0? graph.view.translate.y - newTranslateY + 5:graph.view.translate.y - newTranslateY - 5
    );
   
  });
}
